# Arabic Font Configuration

This file serves as a placeholder for Arabic font support in the PDF generation system.

For production use, please download and place the following fonts in this directory:
- NotoSansArabic-Regular.ttf
- Cairo-Regular.ttf
- Amiri-Regular.ttf

Font sources:
- Google Fonts: https://fonts.google.com/
- Adobe Fonts: https://fonts.adobe.com/

The PDF generator will fall back to system fonts if Arabic fonts are not available.
